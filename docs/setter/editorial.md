# Editorial

## Main Goals

The main goal of this stage is:
- Write an editorial that explains the intended solution clearly.

::: info Checklist for this stage
- Write the solution in the shared Overleaf document.
:::

## Purpose of the Editorial

The editorial is the official solution description for the problem. Its main purpose is to help readers understand the solution.

To write a good editorial, focus on the following three points:

- the intuition behind the key idea or observation;
- a rigorous proof of why the solution works;
- implementation details, if there are tricky parts.

In most cases, the editorial should also include the intended time complexity of the solution.

We recommend writing solutions for every subtask. However, you should mainly focus on subtasks that give useful insight toward the full solution.

## Before Moving On

::: tip Ready to move on when
- The editorial explains the main idea of the intended solution.
- The editorial includes a correctness proof.
- The intended time complexity is written.
- Important implementation details are mentioned, if needed.
:::
